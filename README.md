Simple TOTP device.

Based on:
1. [T-PicoC3](https://github.com/Xinyuan-LilyGO/T-PicoC3) board with [lilygo-micropython](https://github.com/Xinyuan-LilyGO/lilygo-micropython) firmware
3. [st7789](https://github.com/russhughes/st7789_mpy) driver for MicroPython
4. [pico-2fa-totp](https://github.com/eddmann/pico-2fa-totp) repo
5. [DIYables](https://diyables.io) MicroPython library
6. [micropython-time](https://github.com/derekenos/micropython-time) library
